# Personal-life-Manager-v1
A C++ console application to manage daily tasks and goals with menu-driven options.

## Life OS (C++)
A menu-driven productivity system built using Object-Oriented Programming.

### Features
- Task management
- Goal tracking
- Mark tasks complete
- Update tasks and goals
- Beginner-friendly OOP structure
  
### Concepts Used
- Classes & Objects
- Vectors
- Encapsulation
- Menu-driven programming
