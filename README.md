# Personal-life-Manager-v1
A C++ console application to manage daily tasks and goals with menu-driven options.

## Task & Goal Manager (C++)

### Description
This is a beginner-friendly menu driven project built using C++.
It allows users to:
- Add tasks
- Mark tasks complete
- Add goals
- Update deadlines

### Concepts Used
- Classes & Objects
- Vectors
- Menu-driven programming
